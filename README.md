# 基于最短路径问题的交通查询系统 (Traffic Query System)

## 项目简介
本项目为长安大学数据结构与算法综合实验项目。系统模拟了现实世界中的交通网络，将城市视为节点，交通线路视为有向边。系统旨在为用户提供基于图论的最优出行方案规划，支持多维度的交通路线查询与动态数据管理。

## 核心功能
* **多目标路径规划**：支持针对指定的起点和终点，计算并输出三种最优出行方案：最短路径方案、最少费用方案、最短时间方案。
* **详细路径回溯**：不仅输出总里程、总费用和总时间，还能通过路径回溯算法，详细列出途径的中间城市及到达每个节点的具体时刻。
* **动态路径添加**：系统提供 GUI 界面供用户动态录入新路径，并具有完善的防呆验证机制（如：限制城市名为纯英文字母、数值必须为正数、起点终点不能相同等）。
* **数据持久化**：自定义的图数据支持从本地文件 (`traffic_sample.txt`) 动态加载与写入，利用文件流操作保障数据的持久化存储。

## 核心算法与数据结构
* **线段树优化 Dijkstra**：摒弃了传统的线性扫描或二叉堆，自主实现**线段树 (Segment Tree)** 来维护节点集合的距离值，以支持高效的区间查询与单点更新。
* **算法复杂度**：时间复杂度优化至 $O((V+E)\log V)$，在处理特定区间约束或动态权值更新时具有更高效率。
* **字符串哈希与映射**：利用哈希表实现城市名称（字符串）到图节点索引（整型）的快速转换与查询。

## 技术栈
* **开发语言**：C++
* **图形界面**：Qt Creator / Qt Framework
* **数据结构**：图 (Graph)、线段树 (Segment Tree)、邻接表、哈希映射
